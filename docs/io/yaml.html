YAML Interface¶

muspy.load_json(path)[source]

Load a JSON file into a Music object.

Parameters

path (str or Path) – Path to the file to load.

Returns

Loaded Music object.

Return type

muspy.Music

muspy.save_json(path, music, ensure_ascii=False, indent=None)[source]

Save a Music object to a JSON file.

Parameters

  • path (str or Path) – Path to save the JSON file.

  • music (muspy.Music) – Music object to save.

  • ensure_ascii (bool) – Whether to escape non-ASCII characters. Defaults to False. See json.dumps().

  • indent (int or str) – Indent level. See json.dumps().

Note

A YAML schema is available for validating a YAML file against MusPy’s format.
